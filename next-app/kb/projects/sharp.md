Sharp is the unified control panel for the Clawdbot ecosystem.

## Features

- Agent session management
- Knowledge base with [[concepts/wiki-links]]
- Real-time WebSocket connection to Clawdbot gateway

## Stack

- Next.js 14+ with App Router
- TypeScript
- Tailwind CSS
